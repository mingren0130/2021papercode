CVFS

CVFS.py is a Python File can solve classification problem, use machine learning classification method is Support Vector Machine(SVM) 

Python packge

	numpy
	pandas
	xgboost
	sklearn.svm
	sklearn.utils
	sklearn.model_selection
	sklearn.feature_selection
	sklearn.preprocessing

Usage

	python CVFS.py -i <inputfilename>

	Preset parameter

		-s <Number of choices> is 3
		-e <Number of executions> is 3
		-o <XGBClassifier jobs> is 4
		-r <Use machine learning method> is classification
